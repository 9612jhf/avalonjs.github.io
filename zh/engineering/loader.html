baseUrl配置项的使用

avalon1.4的新特性, 有时, javascript文件并不是放在根目录下,我们每个文件就需要写得长, 有了它, 每个可以写很短

例子, 见下面目录结构

主页面的index.html如下:

<!DOCTYPE html>
<html>
    <head lang="en">
        <meta charset="UTF-8">
        <title></title>
        <script type="text/javascript" src="avalon.js"></script>
        <script type="text/javascript">
            require.config({
                baseUrl: "./examples/loader/"
            })
            require(["./aaa"], function(a){
                avalon.define({
                    $id: "test",
                    title: "加载成功"
                })
                avalon.log(a)
                avalon.scan()
            })
        </script>
    </head>
    <body>
        <div ms-controller="test">
            <h1>{{title}}</h1>
        </div>
    </body>
</html>

由于我们设置baseUrl, 当我们加载./aaa模块时, 它的路径就变成./examples/loader/aaa, 再自动加上.js后缀名

我们再看一下所有JS文件是如何写的:

aaa.js

define(["./bbb"], function(){
     avalon.log("已经加载aaa模块")
    return "aaa"
})

bbb.js

define(["./ccc", "./ddd"], function() {
    avalon.log("已经加载bbb模块")
    return "bbb"
})

ccc.js

define(function(){
    avalon.log("已经加载ccc模块")
    return "ccc"
})

ddd.js

define(["./eee"], function() {
    avalon.log("已经加载ddd模块")
    return "ddd"
})

eee.js

define(function() {
    avalon.log("已经加载eee模块")
    return "eee"
})

paths配置项的使用

可以让一些非常长的路径名改成一个单词，然后我们直接在require中引用这些单词就行了。

看下面例子， 这几个例子内部都使用了amd机制，支持三种引入机制。

jquery.placeholder可以在这里下，https://github.com/mathiasbynens/jquery-placeholder

主页面的index.html如下:

<!DOCTYPE html>
<html>
    <head lang="en">
        <meta charset="UTF-8">
        <title></title>
        <script type="text/javascript" src="avalon.js"></script>
        <script type="text/javascript">
            require.config({
                baseUrl: "./lib/",
                paths: {
                    jquery: "jquery-2.1.3",
                    "jquery.placeholder": "jquery.placeholder",
                    underscore: "underscore-1.7",
                    backbone: "backbone"
                }
            })
            require(["backbone", "jquery.placeholder"], function(a) {
                console.log(a, jQuery.fn.placeholder)
                avalon.define({
                    $id: "test",
                    title: "加载成功"
                })
                avalon.scan()
            })
        </script>
    </head>
    <body>
        <div ms-controller="test">
            <h1>{{title}}</h1>
        </div>
    </body>
</html>
